# 字符串的声明
`char *s = "hello, world!"`<br>
`char s[] = "hello,world!"`<br>
`char s[20] = "hello,world!"`

## `char *s = "hello, world!"`
对于这种写法：<br>
`char *s1 = "hello, world!"`<br>
`char *s2 = "hello, world!"`<br>
s1,s2两个指针指向的地址是同一个地址<br>
字符串`"hello, world!"`以字符数组的形式，存放在只读的代码段<br>
不能通过`s[0] = 'b'`这样的语句修改
